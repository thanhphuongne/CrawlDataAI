Buttons

Directive app-button

Properties

Property
Accept value
Description
btn-type
"filled" | "outline" | "subtle" | "text"
Default: "filled". Type style of button
btn-variant
"primary" | "secondary" | "tertiary" | "success" | "danger" | "warning"
Default: "primary". Variant color of button.
btn-size
'sm' | 'md' | 'lg'
Default: "md". Size of button.
btn-quake
boolean
Default: false.
