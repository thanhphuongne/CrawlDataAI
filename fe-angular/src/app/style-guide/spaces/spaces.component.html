Spaces

Base space is: 4px and define with variable: --base-space:4px;

This section will include height, width, border, padding, margin.

Responsive rule

Breakpoint of responsive will follow bootstrap ver 5. Keywords using for breakpoint (res_key): blank, sm, md, lg, xl, xxl

Keyword must add before value. Example: w-sm-100vw

Width

Type
Class
Description
View width
.w-100vw
width: 100vw
Pixel width
.w-px
.w[res_key]-[width]

res_key follow responsive rule, start with - except blank

width start from 0 to 40. With base width is 4px.

Percent width
.w[res_key]-p-[col]
.min-w[res_key]-p-[col]
.max-w[res_key]-p-[col]

res_key follow responsive rule, start with - except blank

col follow rule 12 cols start from 1 to 12

Example:
{{ ".w-p-6{width:50%}" }}
{{ ".w-sm-p-6{width:50%}" }}

Max/min width

.min-w-1

width start from 0 to 100. With base width is 4px.
